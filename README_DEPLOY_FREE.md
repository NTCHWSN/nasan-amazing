# Deploy this app for free with GitHub Pages

This folder is ready to upload to GitHub Pages.

What you need:

- A free GitHub account
- A public repository
- GitHub Pages enabled from the repository settings

Upload rules:

- Upload the contents of this folder, not the folder itself.
- `index.html` must be at the top level of the repository.
- Keep these folders at the top level: `assets`, `icons`, `src`, `vendor`.
- Keep these files at the top level: `manifest.webmanifest`, `service-worker.js`, `offline.html`.

After GitHub Pages is enabled, your app URL will look like:

```text
https://YOUR-GITHUB-USERNAME.github.io/nasan-amazing/
```

Install on iPhone/iPad:

1. Open the URL in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
4. Tap Add.

Install on Android:

1. Open the URL in Chrome.
2. Tap the menu.
3. Tap Install app or Add to Home screen.

Install on computer:

1. Open the URL in Chrome or Edge.
2. Use the install icon in the address bar, or browser menu > Install app.

Offline use:

Open the app once while online and wait for the game to load. The service worker caches the game files for later offline use.
